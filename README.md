## Requirements

- Make sure >= Node 18 is installed.

## Development

### Install the dependencies

```bash
npm install
```

### Run the local Storybook server

```bash
npm run storybook
```

## Deployment

This project is deployed in [Chromatic](https://64c2830027035640b81d0e9c-ynlrpqmjhl.chromatic.com/).

```bash
npm run chromatic
```
